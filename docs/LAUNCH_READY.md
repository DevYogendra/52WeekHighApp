## 🚀 Quick Start Integration

### Easiest Method (2 minutes)

Add to your `streamlit_app.py` sidebar:

```python
with st.sidebar:
    st.divider()
    st.subheader("📚 Help & Reference")
    
    docs_path = Path(__file__).parent / "docs"
    st.write(f"[🚀 Quick Start](file:///{docs_path / 'quick-start.html'})")
    st.write(f"[📖 How to Read](file:///{docs_path / 'interpretation-guide.html'})")
    st.write(f"[📚 Glossary](file:///{docs_path / 'glossary.html'})")
    st.write(f"[🏠 All Docs](file:///{docs_path / 'index.html'})")
```

**Result:** 4 clickable links in sidebar that open HTML pages in browser tabs

### Better Method (3 minutes)

Use the included Python helper:

```python
from pathlib import Path
from docs.html_integration import DocumentationHelper

docs = DocumentationHelper()
docs.add_sidebar_navigation()
```

**Result:** Professional-looking Help section with buttons and icons

### Troubleshooting

**Links not working?**
- Check file paths are correct
- Use absolute paths for file:// URLs
- Verify files exist in docs/ directory

**Styling looks wrong?**
- Check styles.css is in same directory
- Clear browser cache (Ctrl+Shift+Del)
- Check file permissions

**Performance slow?**
- Files are ~40KB each (should be instant)
- Check network connection
- Try local method if using remote server
